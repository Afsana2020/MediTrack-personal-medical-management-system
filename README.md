# MediTrack - Personal Medical Management System

## Overview

MediTrack is a comprehensive personal medical management system that helps patients and caregivers track medications, appointments, symptoms, and prescriptions in one place.

##  Features

- **Medicine Tracking** - Track medications, dosages, and refill reminders
- **Appointment Scheduler** - Manage doctor appointments with reminders
- **Prescription Management** - Upload and organize prescriptions
- **Symptom Journal** - Log symptoms and track health patterns
- **Caregiver Access** - Share health data with caregivers
- **Smart Notifications** - Get alerts for missed doses and appointments

## Tech Stack

**Frontend:**
- React.js
- React-Bootstrap
- Axios
- React Router DOM

**Backend:**
- Node.js
- Express.js
- MongoDB 


## Installation

### Prerequisites
- Node.js installed
- MongoDB installed locally or MongoDB Atlas account

### Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/Afsana2020/MediTrack-personal-medical-management-system.git
   cd MediTrack-personal-medical-management-system```


2. **Install backend dependencies**
   ```bash
   cd server
   npm install```
3. **Install frontend dependencies**
   ```bash
   cd ../client
   npm install
   ```

4. **Set up environment variables**
Create a .env file in the server folder:
```
PORT=5000
MONGODB_URI=your_mongodb_connection_string
```

5. **Run the application**

- Backend (from server folder):
```bash
npm run start
```

- Frontend (from client folder):
```bash
npm start
```

- Open your browser and go to ```http://localhost:3000```
